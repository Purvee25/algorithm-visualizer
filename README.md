# Algorithm Visualizer

Interactive sorting and graph algorithm visualizations built with Python.

## Algorithms
- Sorting: Bubble, Quick, Merge, Heap, Insertion
- Graph: BFS, DFS, Dijkstra, A*
- Tree: BST operations, AVL rotations

## Usage
```python
from visualizer import SortVisualizer
sv = SortVisualizer()
steps = sv.quicksort([5, 3, 8, 1, 9, 2])
```
